# Laravel product catalog test

Laravel 12 + VueJS + PostgreSQL + Docker

## Run Demo

Clone the project

```bash
  git clone https://github.com/vlasovim/laravel-product-catalog-test
```

Go to the project directory

```bash
  cd laravel-product-catalog-test
```

Start (see Makefile)

```bash
  sudo make start
```

Stop (see Makefile)

```bash
  sudo make stop
```

After starting, access the application at: http://localhost:8085/. Each start performs a fresh installation.
